# Aave V3 en DeFi — Material de clase

Repositorio con el material de la primera clase sobre el protocolo **Aave V3** dentro del ecosistema DeFi.

## Archivos

| Archivo | Descripción |
|---|---|
| `presentacion.html` | Presentación interactiva (22 slides) |
| `glosario.html` | Glosario de términos DeFi y Aave |
| `assets/` | Imágenes y logos |

## Cómo usar

Abre directamente `presentacion.html` en el navegador — no requiere servidor ni dependencias.

Navega con las flechas del teclado `←` `→` o con los botones de la barra inferior.

## Temas cubiertos

- Flujos de **supply**, **borrow**, **repay** y **withdraw**
- **aTokens** y **debt tokens**
- Token de gobernanza **AAVE** y stablecoin nativa **GHO**
- Isolation Mode, E-Mode y portales cross-chain
